## 🛠️ Proyectos Actuales

### 🐉 1. Simulador de Física de Cola (Dragon Cursor)
Inspirado en el arte generativo y cinemática inversa. Es un script que simula el movimiento fluido de un dragón en la terminal usando física de interpolación.
* **Conceptos:** Bucles de eventos, `msvcrt`, interpolación lineal.
* **Ejecución:** `python dragon_cursor.py`

### 🎮 2. Gamer Stat Tracker
Un sistema de progresión de niveles basado en el rendimiento de partidas, ideal para entender cómo funcionan las bases de datos locales.
* **Conceptos:** Diccionarios, manejo de estados, lógica de XP.
* **Ejecución:** `python gamer_stats.py`

### 🔐 3. Suite de Ciberseguridad Básica
Dos herramientas fundamentales para entender la protección de datos:
* **Generador de Claves:** Crea contraseñas seguras y aleatorias.
* **Cifrado César:** Encripta mensajes secretos moviendo posiciones en el abecedario.
* **Ejecución:** `python simulador.py` / `python cifrado.py`
